\# 🖥️ Connection With Database in C# (ADO.NET)



This repository contains an educational project demonstrating how to establish a secure and efficient connection between a \*\*C# Application\*\* and a \*\*SQL Server Database\*\* using \*\*ADO.NET (Connected Architecture)\*\*.



\## 🚀 Features \& Concepts Covered

\* \*\*SqlConnection\*\*: Managing opening and closing database sessions safely.

\* \*\*SqlCommand\*\*: Executing SQL Queries and Stored Procedures.

\* \*\*SqlDataReader\*\*: Fast, forward-only retrieval of data rows from the database.

\* \*\*Data Sanitization\*\*: Preventing SQL Injection attacks using parameterized queries.

\* \*\*Configuration Management\*\*: Storing connection strings securely in `App.config`.



\## 📚 Curriculum Context

This project is part of my structured software engineering advancement under the \*\*Abu-Hadhoud Programming Roadmap (Course 18)\*\*, focusing on database development and backend mechanics in C#.



\## ⚙️ How to Run

1\. Clone the repository.

2\. Update the connection string in `App.config` to match your local SQL Server instance.

3\. Build and run the project using Visual Studio.
